## ✅ PASSO 2: Ativar GitHub Pages

1. No menu à esquerda, procure por **"Pages"**
2. Clique em **"Pages"**

Você verá uma tela assim:

```
GitHub Pages
Source: 

[Dropdown] None
            main
            gh-pages
            docs folder
```

3. **Clique no dropdown "None"**
4. **Selecione "main"** (nossos arquivos estão aqui)
5. **Clique em "Save"**

2. Atualize a página (F5)

## ❌ Se não funcionar?

### Problema: "404 - Página não encontrada"
- [ ] Aguarde 5 minutos (pode levar mais tempo)
- [ ] Atualize o navegador (Ctrl+F5)
- [ ] Verifique se "Source" está em "main branch"

### Problema: GitHub Pages não aparece
- [ ] Clique em Settings → Pages
- [ ] Confirme que Source está em "main"
- [ ] Confirme que você salvou as mudanças

### Problema: Arquivos não aparecem
- [ ] Vá a https://github.com/nataliavogt/sleepcare-pro
- [ ] Verifique se os arquivos .html estão visíveis
- [ ] Se há muitos arquivos não-commitados, é seguro deletá-los (deixar só os principais)
