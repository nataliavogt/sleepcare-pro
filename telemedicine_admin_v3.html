🎨 Guia de Customização

✓ Fácil de Customizar: Você pode mudar cores, fontes e texto sem conhecimento técnico

1. Mudar Cores Principais

Como fazer:

  1. Abra o arquivo HTML em um editor de texto
  2. Procure por: #667eea
  3. Substitua por sua cor (usando Color Picker)

Cores recomendadas: Verde (#10b981), Azul (#0b63e5), Teal (#0d9488)

2. Mudar Nome da Clínica

Procure por: SleepCare Pro

Substitua por: Seu nome (ex: "Clínica Natália Vogt")

3. Mudar Fontes

Passo 1: Visite Google Fonts

Passo 2: Escolha uma fonte (recomendadas: Poppins, Inter, Raleway)

Passo 3: Cole o código no HTML e substitua font-family

4. Adicionar Logo

Passo 1: Hospede sua logo em uma URL (Imgur, Google Drive, etc)

Passo 2: Procure por <h1>💤 SleepCare Pro</h1>

Passo 3: Substitua por:

<img src="sua-url-da-logo.png" height="40"> <h1 style="display: inline;">SleepCare Pro</h1>

5. Arquivo de Guia Completo

📚 Documentação Disponível: Verifique o arquivo CUSTOMIZATION_GUIDE.md para guia detalhado de todas as alterações possíveis

Dicas Rápidas

  • Backup: Sempre faça uma cópia do arquivo antes de editar
  • F12: Use DevTools (F12) para testar alterações em tempo real
  • Procurar/Substituir: Use Ctrl+H no editor para encontrar/substituir rápido
  • Ícones: Todos são emojis, muito fáceis de mudar!
