# 🛏️ SleepCare Pro - Plataforma de Medicina do Sono

Plataforma web moderna para pacientes da Dra. Natália Vogt com questionários validados de medicina do sono.

## ✨ Funcionalidades

- ✅ Autenticação segura com Firebase (Email + CPF)
- ✅ 4 Questionários validados (STOP-BANG, Epworth, PSQI, ISI)
- ✅ Diário de sono estruturado em português
- ✅ Dashboard de progresso com histórico
- ✅ Integração WhatsApp para reminders
- ✅ Envio automático de scores por email
- ✅ Conformidade LGPD + TCLE

## 🚀 Tecnologia

- **Frontend**: Next.js 14 + React 18
- **Backend**: Node.js (API Routes)
- **Database**: Firebase Firestore
- **Hospedagem**: Vercel
- **Estilo**: Tailwind CSS

## 📋 Setup

```bash
npm install
npm run dev
# Acesse http://localhost:3000
```

Edite `.env.local` com suas credenciais do Firebase.
