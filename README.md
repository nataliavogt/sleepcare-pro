# SleepCare Pro - Plataforma de Questionários de Medicina do Sono

Uma plataforma web completa para pacientes responderem questionários validados de medicina do sono com registro seguro, conformidade com LGPD e painel administrativo integrado.

## 🎯 Características

### Para Pacientes
- **Registro Seguro**: Cadastro com Nome Completo, Email, Telefone e CPF
- **Validação de CPF**: Validação de dígitos verificadores em tempo real
- **4 Questionários Validados**:
  - STOP-Bang (triagem de apneia obstrutiva do sono)
  - PSQI (índice de qualidade do sono de Pittsburgh)
  - ISI (escala de gravidade da insônia)
  - ESE (escala de sonolência de Epworth)
- **Interface Mobile-First**: Uma pergunta por tela
- **Scoring Automático**: Pontuação e interpretação instantânea
- **LGPD Compliant**: Consentimento e política de privacidade
- **Histórico**: Acompanha todas as respostas e resultados

### Para Administrador
- **Painel de Controle**: Dashboard com estatísticas
- **Gerenciar Pacientes**: Visualizar lista de registros
- **Ver Resultados**: Acessar todos os questionários
- **Personalizar Plataforma**:
  - Upload de logo
  - Customizar mensagens
  - Ajustar cores

## 🌐 GitHub Pages Deployment

### Passo 1: Criar Repositório no GitHub
1. Vá para https://github.com/new
2. **Repository name**: `sleepcare-pro`
3. Deixe **Public** marcado
4. Clique **Create repository**

### Passo 2: Upload dos Arquivos
1. Clique **Add file > Upload files**
2. Arraste ou selecione:
   - `sleepcare_questionnaire_platform.html`
   - `admin-panel.html`
   - `README.md`
3. Clique **Commit changes**

### Passo 3: Ativar GitHub Pages
1. Vá para **Settings**
2. Clique em **Pages** (no menu lateral)
3. Em **Source**, selecione **main branch**
4. Clique **Save**

### Resultado
Seu site estará em:
```
https://nataliavogt.github.io/sleepcare-pro/
```
