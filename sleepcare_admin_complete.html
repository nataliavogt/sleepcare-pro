Bem-vindo ao Painel Administrativo

Este é seu espaço para gerenciar pacientes, aprovar novos acessos e acompanhar o uso da plataforma SleepCare Pro. Aqui você tem controle total sobre:

  • Pacientes: Visão completa de todos os seus pacientes registrados
  • Aprovações: Revisar e aprovar novo acesso para pacientes
  • Email: Configurar integração com EmailJS para envio automático
  • Auditoria: Rastrear todas as ações para LGPD compliance
  • Configurações: Personalizar dados da clínica

📌 Informações Importantes

🔐 Segurança: Todos os dados dos pacientes são armazenados localmente e protegidos.

✅ LGPD: Esta plataforma está em conformidade com a Lei Geral de Proteção de Dados Pessoais.

📊 Privacidade: Nenhum dado é compartilhado com terceiros sem consentimento explícito.

📧 Configuração de Email

Configure o EmailJS para enviar automaticamente emails de aprovação para pacientes.

📚 Como Configurar EmailJS:

  1. Acesse emailjs.com e crie uma conta gratuita
  2. Configure um serviço de email (Gmail, Outlook, etc)
  3. Crie um template de email para aprovação
  4. Copie suas credenciais nos campos acima
  5. Clique em "Salvar Configurações"
